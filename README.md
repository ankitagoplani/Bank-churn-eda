
# 🏦 Bank Customer Churn — EDA & Analysis

## Problem Statement
Analyse 10,000 bank customers to identify key drivers of churn.

## Key Findings
- 20.4% churn rate — imbalanced dataset
- Age is strongest predictor (older = more churn)
- Germany has 2x higher churn than France/Spain
- Inactive members churn significantly more
- Zero balance customers show unusual churn pattern

## Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Scipy

## Dataset
Kaggle — Bank Customer Churn Modelling
